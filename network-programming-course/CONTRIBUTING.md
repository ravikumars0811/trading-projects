# Contributing to Network Programming Course

Thank you for your interest in contributing to this course!

## How to Contribute

### Reporting Issues

- Use GitHub Issues to report bugs or suggest improvements
- Provide detailed information about the issue
- Include code examples if applicable

### Adding Examples

1. Fork the repository
2. Create a new branch for your example
3. Add your code to the appropriate module's `examples/` directory
4. Update the module's README.md with a link to your example
5. Add compilation instructions
6. Submit a pull request

### Improving Documentation

- Fix typos, clarify explanations
- Add diagrams or visualizations
- Provide additional resources
- Translate content

### Code Style

- Use C++17 or later
- Follow existing code style
- Add comments for complex sections
- Include compilation and usage instructions
- Test your examples before submitting

### Example Template


## Code of Conduct

- Be respectful and inclusive
- Provide constructive feedback
- Help others learn
